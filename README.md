# HealthCareAnalysis_by_Sinjini

This project presents an interactive Healthcare Claims Analysis Dashboard created using Power BI.
The dashboard helps healthcare administrators and insurance analysts gain insights into patient claims, insurance payments, provider performance, demographics, and care metrics.

Project Goal

Deliver deep insights on hospital claims, insurance payments, and patient demographics to help optimize financial planning & healthcare operations.

Key Highlights

1.Insurance vs. patient cost distribution
2.Age & gender-based patient analysis
3.Provider performance & top claim contributors
4.Disease type frequency and medical stay cost patterns
5.Interactive slicers for rich user experience
